# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus criticism on code and ideas, not people.
- Assume good intent and communicate clearly.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Trolling, deliberate provocation, or intimidation
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

For serious issues, contact the maintainer directly through GitHub.
